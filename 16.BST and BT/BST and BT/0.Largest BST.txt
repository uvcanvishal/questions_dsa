Assignment Coding Problem
Problem Name: Largest BST
Problem Level: MEDIUM
Problem Description: ####Given a Binary tree, find the largest BST subtree. That is, you need to find the BST with maximum height in the given binary tree.

#####Return the height of largest BST.

 `Input format : `

`Line 1 : Elements in level order form (separated by space)`

`(If any node does not have left or right child, take -1 in its place)`

 #####Sample Input 1: 
    5 10 6 2 3 -1 -1 -1 -1 -1 9 -1 -1

#####Sample Output 1: 
    2
