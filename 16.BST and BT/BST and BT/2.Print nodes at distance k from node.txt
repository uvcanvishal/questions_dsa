Assignment Coding Problem
Problem Name: Print nodes at distance k from node
Problem Level: MEDIUM
Problem Description: ####Given a binary tree, a node and an integer K, print nodes which are at K distance from the the given node. 

 `Input format : `

`Line 1 : Elements in level order form (separated by space)`

`(If any node does not have left or right child, take -1 in its place)`

`Line 2 : Node`

`Line 3 : K`

`Output format : Answer nodes in different line`

 #####Sample Input : 
    5 6 10 2 3 -1 -1 -1 -1 -1 9 -1 -1
    3
    1

#####Sample Output : 
    9
    6
