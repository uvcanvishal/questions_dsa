Assignment Coding Problem
Problem Name: Pattern Matching
Problem Level: MEDIUM
Problem Description: ####Given a  list of n words and a pattern p that we want to search. Check if the pattern p is present the given words or not.
####Return true or false.

#####Input Format : 
    Line 1 : Integer n
    Line 2 : n words (separated by space)
    Line 3 : Pattern p (a string)

#####Output Format :
    true ot false
 #####Sample Input 1 :
    4
    abc def ghi cba
    de

#####Sample Output 2 : 
    true

#####Sample Input 2 :
    4
    abc def ghi hg
    hi

#####Sample Output 2 : 
    true

#####Sample Input 3 :
    4
    abc def ghi hg
    hif

#####Sample Output 3 : 
    false
