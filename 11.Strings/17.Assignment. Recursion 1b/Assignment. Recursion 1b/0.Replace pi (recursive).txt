Problem Name: Replace pi (recursive)
Problem Level: MEDIUM
Problem Description: ####Given a string, compute recursively a new string where all appearances of "pi" have been replaced by "3.14".
 #####Sample Input 1 :
    xpix

#####Sample Output :
    x3.14x

#####Sample Input 2 :
    pipi

#####Sample Output :
    3.143.14

#####Sample Input 3 :
    pip

#####Sample Output :
    3.14p
