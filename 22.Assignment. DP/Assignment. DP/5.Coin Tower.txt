Assignment Coding Problem
Problem Name: Coin Tower
Problem Level: MEDIUM
Problem Description: ####Whis and Beerus are playing a new game today . They form a tower of N coins and make a move in alternate turns . Beerus being the God plays first . In one move player can remove 1 or X or Y coins from the tower . The person to make the last move wins the Game . Can you find out who wins the game ?

#####Input Format :
    Contains three value N,X,Y as mentioned in the problem statement

#####Output Format :  
    A string containing the name of the winner like “Whis” or “Beerus” (without quotes)

#####Constraints:
    1<=N<=10^6
    2<=X<=Y<=50
 #####Sample Input :
     4 2 3

#####Sample Output :
     Whis
