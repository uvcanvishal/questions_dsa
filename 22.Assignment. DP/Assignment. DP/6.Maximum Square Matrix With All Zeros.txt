Assignment Coding Problem
Problem Name: Maximum Square Matrix With All Zeros
Problem Level: MEDIUM
Problem Description: ####Given a n*m  matrix which contains only 0s and 1s, find out the size of maximum square sub-matrix with all 0s. You need to return the size of square with all 0s.

#####Input format :
    Line 1 : n and m (space separated positive integers)
    Next n lines : m elements of each row (separated by space).

#####Output Format:
    Line 1 : Size of maximum square sub-matrix






 #####Sample Input : 
    3 3
    1 1 0
    1 1 1
    1 1 1

#####Sample Output :
    1
