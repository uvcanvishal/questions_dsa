Coding Problem
Problem Name: BST to LL
Problem Level: EASY
Problem Description: ####Given a BST, convert it into a sorted linked list. Return the head of LL.

#####Input format : 
    Elements in level order form (separated by space)
    (If any node does not have left or right child, take -1 in its place)

#####Output Format : 
    Linked list elements (separated by space)

 #####Sample Input : 
    8 5 10 2 6 -1 -1 -1 -1 -1 7 -1 -1

#####Sample Output : 
    2 5 6 7 8 10
